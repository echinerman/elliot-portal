Frequently Asked Questions

How do I set up my Firestick?
We recommend using TiviMate Premium for Firestick. You can find step-by-step instructions in the tab.
Why is my stream buffering?
Buffering is usually caused by local internet issues.
  • Restart your router and device.
  • Ensure you are using the Cloudflare Optimized server URL found in your Dashboard.
  • If using WiFi, try moving closer to the router or using a hardwired ethernet connection.
Can I watch on multiple TVs at once?
A standard license allows for one connection at a time. If you want to watch on two TVs simultaneously (e.g., Living Room and Bedroom), you will need to purchase an additional license. Please contact Elliot to upgrade.
What is EPGenius?
EPGenius (GanjaRelease #6) is a smart M3U Playlist link. Unlike the raw link from the server, EPGenius automatically organizes channels better and includes a high-quality Electronic Program Guide (EPG) built-in. Always use this link in your player if possible.

Setup Documentation

Follow these guides to connect your device.

TiviMate Premium (Recommended)

Step 1: Install TiviMate

  1. Install the Downloader app from the Amazon App Store.
  2. Enter code: 278077 (Official TiviMate) to download.
  3. Follow prompts to install.

Step 2: Premium & Dual Playlist Setup

To get the best experience (Live TV + VOD), we use two separate lists.

List 1: Live TV (EPGenius)

  1. Settings > Playlists > Add Playlist. Select M3U Playlist.
  2. Enter your EPGenius M3U URL (from Dashboard).
  3. Name it "Live TV".
  4. Important: Manage groups > Hide all VOD groups (Movies/Series).

List 2: VOD (Movies/Series)

  1. Add Playlist > Xtream Codes.
  2. Enter Server, User, Pass from Dashboard. Check "Include VOD".
  3. Name it "VOD".
  4. Important: Manage groups > Hide all Live TV groups.

UHF (Paid) or Chillio

  • Use Xtream Codes login from Dashboard.
  • Enter Server URL, User, Pass.
